# GraphPad Prism Quick Cheat Sheet

## 1) Standard curve / Prediction
- Use an **XY table**
- X = known concentration
- Y = measured signal
- Run **Linear regression** or **Interpolate a standard curve**

## 2) Two-group comparison
- Use a **Column table**
- **Unpaired**: different subjects/samples in each group
- **Paired**: same subject measured twice or matched pairs
- Typical tests: unpaired t, Welch t, paired t, or nonparametric alternative

## 3) Three or more groups
- Use a **Column table** for one-factor independent groups
- Typical test: **One-way ANOVA**
- If needed, follow with multiple comparisons

## Graphing tip
Show **individual points** whenever possible.
