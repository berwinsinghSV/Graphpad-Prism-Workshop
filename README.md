# GraphPad Prism Basics Workshop

Welcome to the teaching materials for **GraphPad Prism Basics for Biomedical Data Analysis**.

This public repository contains only **student-facing teaching files**:
- the slide deck,
- practice datasets,
- a quick-start guide,
- and a simple Prism workflow cheat sheet.

---

## Start Here

### 1) Download the slide deck
Open `slides/GraphPad_Prism_Basics_Workshop_Berwin_Singh.pptx`

### 2) Download the datasets
Use the files in `datasets/`.

### 3) Open GraphPad Prism
Follow the examples in this order:
1. `irinotecan_standards.csv`
2. `irinotecan_unknowns.csv`
3. `rat_bp_unpaired.csv`
4. `rat_bp_paired.csv`
5. `h9c2_cell_death_5groups.csv`

---

## Workshop Examples

### Example 1 — Standard curve / regression
- Build an Irinotecan standard curve
- Use an **XY table**
- Interpolate unknown sample concentrations

### Example 2 — Two-group comparisons
- Rat systolic blood pressure
- Compare **unpaired** vs **paired** study design
- Use **Column tables** and learn when to choose the appropriate t test

### Example 3 — Three or more groups
- H9C2 cell death study
- Use a **Column table**
- Run a **one-way ANOVA** for independent groups
